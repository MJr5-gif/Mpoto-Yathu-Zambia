# Mpoto Yathu Zambia 

A professional restaurant landing page built for **Mpoto Yathu Zambia** — a traditionally inspired Zambian Matebeto and Grill based in Lusaka, Zambia. Rated 4.6 on Google with 497+ reviews.

## About The Project
This is a client mock-up landing page designed to give Mpoto Yathu Zambia a strong online presence. Built with pure HTML5 and CSS3 — no frameworks, no libraries, just clean semantic code.

## Features
- **Hero slideshow** — pure CSS background animation cycling through real food photos
- **Menu section** — real dishes with prices and photos from their Google listing
- **Real customer reviews** — sourced directly from their Google reviews
- **Google Maps embed** — pointing to their exact verified location
- **Reservation form** — with date picker, guest count, and special requests
- **Fully responsive** — works on mobile, tablet, and desktop
- **Accessible** — semantic HTML, alt text, reduced motion support

## Design
- **Theme:** Deep earthy browns, warm amber/orange, and cream/white
- **Gradients:** Linear and radial gradients throughout for warmth and depth
- **Typography:** Playfair Display (headings) + Lato (body)
- **Animations:** CSS hero slideshow, hover lifts, scroll reveals, nav underlines

## Built With
- HTML5 (semantic markup)
- CSS3 (gradients, keyframe animations, flexbox, grid)
